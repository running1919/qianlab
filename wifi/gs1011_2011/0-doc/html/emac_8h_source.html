00032 /** \addtogroup emac_module
00033  * @{
00034  * Provides the interface to configure and use the EMAC peripheral.
00035  *
00036  * \section emac_usage Usage
00037  * - Configure Emac::EMAC_NCFG with EMAC_Configure(), some of related controls
00038  *   are also available, such as:
00039  *   - EMAC_SetSpeed(): Setup EMAC working clock.
00040  *   - EMAC_FullDuplexEnable(): Working in full duplex or not.
00041  *   - EMAC_CpyAllEnable(): Copying all valid frames (\ref EMAC_NCFG_CAF).
00042  *   - ...
00043  * - Setup Emac::EMAC_NCR with EMAC_NetworkControl(), more related controls
00044  *   can modify with:
00045  *   - EMAC_ReceiveEnable(): Enable/Disable Rx.
00046  *   - EMAC_TransmitEnable(): Enable/Disable Tx.
00047  *   - EMAC_BroadcastDisable(): Enable/Disable broadcast receiving.
00048  *   - ...
00049  * - Manage EMAC interrupts with EMAC_EnableIt(), EMAC_DisableIt(),
00050  *   EMAC_GetItMask() and EMAC_GetItStatus().
00051  * - Manage EMAC Tx/Rx status with EMAC_GetTxStatus(), EMAC_GetRxStatus()
00052  *   EMAC_ClearTxStatus() and EMAC_ClearRxStatus().
00053  * - Manage EMAC Queue with EMAC_SetTxQueue(), EMAC_GetTxQueue(),
00054  *   EMAC_SetRxQueue() and EMAC_GetRxQueue(), the queue descriptor can define
00055  *   by \ref sEmacRxDescriptor and \ref sEmacTxDescriptor.
00056  * - Manage PHY through EMAC is performed by
00057  *   - EMAC_ManagementEnable(): Enable/Disable PHY management.
00058  *   - EMAC_PHYMaintain(): Execute PHY management commands.
00059  *   - EMAC_PHYData(): Return PHY management data.
00060  *   - EMAC_IsIdle(): Check if PHY is idle.
00061  * - Setup EMAC parameters with following functions:
00062  *   - EMAC_SetHash(): Set Hash value.
00063  *   - EMAC_SetAddress(): Set MAC address.
00064  * - Enable/Disable EMAC transceiver clock via EMAC_TransceiverClockEnable()
00065  * - Switch EMAC MII/RMII mode through EMAC_RMIIEnable()
00066  *
00067  * For more accurate information, please look at the EMAC section of the
00068  * Datasheet.
00069  *
00070  * \sa \ref emacd_module
00071  *
00072  * Related files:\n
00073  * emac.c\n
