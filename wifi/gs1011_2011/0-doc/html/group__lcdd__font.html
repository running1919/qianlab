Detailed Description

Purpose

The lcd_font.h files declares a font structure and a LCDD_DrawChar() function that must be implemented by a font definition file to be used with the LCDD_DrawString() method of draw.h.

The font10x14.c implements the necessary variable and function for a 10x14 font.

Note:
Before drawing fonts, canvas should be selected via LCDD_SelectCanvas(), or created by LCDD_CreateCanvas().

Usage

  1. Declare a gFont global variable with the necessary Font information.
  2. Implement an LCDD_DrawChar() function which displays the specified character on the LCD.
  3. Select or create canvas via LCDD_SelectCanvas() or LCDD_CreateCanvas().
  4. Use the LCDD_DrawString() method defined in draw.h to display a complete string.

LCD Driver, LCD Driver Simple Drawing.
