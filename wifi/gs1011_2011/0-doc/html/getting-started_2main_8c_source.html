00034  *  \section Purpose
00035  *
00036  *  The Getting Started example will help new users get familiar with Atmel's
00037  *  AT91 family of microcontrollers. This basic application shows the startup
00038  *  sequence of a chip and how to use its core peripherals.
00039  *
00040  *  \section Requirements
00041  *
00042  *  This package can be used with AT91SAM9xx5 evaluation kits.
00043  *
00044  *  \section Description
00045  *
00046  *  The demonstration program makes two LEDs on the board blink at a fixed rate.
00047  *  This rate is generated by using Time tick timer. The blinking can be stopped
00048  *  using two buttons (one for each LED). If there is no button on board, please
00049  *  type "1" or "2" in the terminal application on PC to control the LEDs 
00050  *  instead.
00051  *
00052  *  \section Usage
00053  *
00054  *  -# Build the program and download it inside the evaluation board. Please
00055  *     refer to the
00056  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/6421B.pdf">
00057  *     SAM-BA User Guide</a>, the
00058  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00059  *     GNU-Based Software Development</a>
00060  *     application note or to the
00061  *     <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00062  *     IAR EWARM User Guide</a>,
00063  *     depending on your chosen solution.
00064  *  -# On the computer, open and configure a terminal application
00065  *     (e.g. HyperTerminal on Microsoft Windows) with these settings:
00066  *    - 115200 bauds
00067  *    - 8 bits of data
00068  *    - No parity
00069  *    - 1 stop bit
00070  *    - No flow control
00071  *  -# Start the application.
00072  *  -# Two LEDs should start blinking on the board. In the terminal window, the
00073  *     following text should appear (values depend on the board and chip used):
00074  *     \code
00075  *      -- Getting Started Example xxx --
00076  *      -- AT91xxxxxx-xx
00077  *      -- Compiled: xxx xx xxxx xx:xx:xx --
00078  *     \endcode
00079  *  -# Pressing and release button 1 or type "1" in the terminal application on 
00080  *     PC should make the first LED stop & restart blinking.
00081  *     Pressing and release button 2 or type "1" in the terminal application on 
00082  *     PC should make the other LED stop & restart blinking.
00083  *
00084  *  \section References
