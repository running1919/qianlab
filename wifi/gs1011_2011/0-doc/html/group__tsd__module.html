Detailed Description

The driver implements touchscreen monitor via TSADC operations. It also provides calibration routine for touchscreen to work with LCD.

Purpose

This unit provides a very powerful touchscreen driver which handles all the complexity. This includes touchscreen calibration, retrieving measurements, configuring the TSADC, etc.

Usage

  1. Implement ADC interrupt handler in application, to invoke TSD_Handler() to handle ADC sampling events for touchscreen monitor.
  2. Call TSD_Initialize() to initialize ADC used for touchscreen.
  3. Call TSD_Calibrate() to do touchscreen calibration with LCD, and enable touchscreen monitor if calibration success.
  4. Call TSD_Enable() to enable or disable touchscreen monitoring.
  5. Declare a global TSD_PenPressed() function anywhere in your code. This function will get called every time the pen is pressed on the screen.
  6. Declare a global TSD_PenMoved() function, which will get called whenever the pen stays in contact with the screen but changes position.
  7. Declare a global TSD_PenReleased() function, which will be invoked as the pen is lifted from the screen.
