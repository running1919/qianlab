Detailed Description

Contents

The directory can basically be divided into several sections for their usage:

  • ADC: the Analog-to-Digital Converter (ADC) controller.
  • AIC: the Advanced Interrupt Controller (AIC) controller.
  • DMAC: the DMA controller.
  • EMAC: the Ethernet MAC 10/100 (EMAC) controller.
  • HSMCI: the High Speed MultiMedia Card Interface (HSMCI) controller.
  • PIO: the Parallel Input/Output Controller (PIO).
  • PIT: the Periodic Interval Timer (PIT) peripheral.
  • PMC: the Power Management (PMC) controller.
  • PWMC: the Pulse Width Modulation controller (PWMC).
  • RTC: the Real-time Clock (RTC) controller.
  • SPI: the Serial Peripheral Interface (SPI).
  • SSC: the Synchronous Serial (SSC) controller.
  • TC: the Timer Counter (TC).
  • TWI: the Two-wire Interface (TWI).
  • USART: the Universal Synchronous Asynchronous Receiver Transceiver (USART) peripheral.
  • LCDC: the LCD Controller (LCDC).
  • WDT: the Interface for the Watchdog Timer (WDT).
