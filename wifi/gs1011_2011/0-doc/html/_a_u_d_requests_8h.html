Detailed Description

Definitions and classes for USB Audio class requests.

  • For a USB device:
    1. After receiving an Audio request from the host, use AUDGenericRequest_GetEntity to determine the target Unit or Terminal.
    2. After receiving an Audio request from the host, use AUDGenericRequest_GetInterface to know which interface is referenced.
    3. Use AUDFeatureUnitRequest_GetControl to determine which control shall be tweaked following a host request.
    4. Use AUDFeatureUnitRequest_GetChannel to identify the channel which will have its control altered.
