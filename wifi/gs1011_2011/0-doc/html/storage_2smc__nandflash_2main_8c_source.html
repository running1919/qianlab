00030 /**
00031  * \page smc_nandflash SMC NAND Flash Example
00032  *
00033  * \section Purpose
00034  *
00035  * This basic nandflash example shall show how to read and write data from/to
00036  * a nandflash connected to the SMC, taking ECC and Bad Block marking into account.
00037  *
00038  * \section Requirements
00039  *
00040  * This package can be used with SAM9xx5 evaluation kits.
00041  *
00042  * \section Description
00043  *
00044  * The required steps are:
00045  * <ul>
00046  * <li> Configure the SMC to interface with the NAND Flash. </li>
00047  * <li> Measure throughtput for RAW data write/read with or without DMA enabled </li>
00048  * <li> Measure throughtput for write/read with or without DMA enabled using software Ecc </li>
00049  * <li> Measure throughtput for write/read with or without DMA enabled using PMECC </li>
00050   * </ul>
00051  * \section Usage
00052  *
00053  * -# Build the program and download it inside the evaluation board. Please
00054  *    refer to the
00055  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/6421B.pdf">
00056  *    SAM-BA User Guide</a>, the
00057  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00058  *    GNU-Based Software Development</a> application note or to the
00059  *    <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00060  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00061  * -# On the computer, open and configure a terminal application
00062  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00063  *   - 115200 bauds
00064  *   - 8 bits of data
00065  *   - No parity
00066  *   - 1 stop bit
00067  *   - No flow control
00068  * -# Start the application.
00069  * -# In the terminal window, the following text should appear:
00070  *     \code
00071  *     -- SMC NandFlash Example xxx --
00072  *     -- xxxxxx-xx
00073  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00074  *     ----
00075  *     - i: Dump Nand flash information
00076  *     - d: Enable or disable DMA
00077  *     - r: Performance test (Raw without ECC) 
00078  *     - s: Performance test (Software ECC)
00079  *     - p: Performance test (PMECC)
00080  *      - h: Display this menu
00081  *    \endcode
00082  *
00083  * \section References
