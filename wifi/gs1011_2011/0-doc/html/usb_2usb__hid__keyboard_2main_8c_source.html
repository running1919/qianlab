00029 /** \cond usb_hid_keyboard
00030  * \page usb_hid_keyboard USB HID Keyboard Example
00031  *
00032  * \section Purpose
00033  *
00034  * The USB HID Keyboard Example will help you to get familiar with the
00035  * USB Device Port(UDP) and PIO interface on AT91SAM microcontrollers. Also
00036  * it can help you to be familiar with the USB Framework that is used for
00037  * rapid development of USB-compliant class drivers such as USB Humen
00038  * Interface Device class (HID).
00039  *
00040  * \section Description
00041  *
00042  * The demo simulates a simple keyboard with a capslock and 'a' on it.
00043  *
00044  * When an EK running this program connected to a host (PC for example), with
00045  * USB cable, the EK appears as a HID Keyboard for the host. Then you can use
00046  * the push buttons on the EK to input letter to the host. E.g, to open a
00047  * editor and input a letter 'a'.
00048  *
00049  * \section Usage
00050  *
00051  * -# Build the program and download it inside the evaluation board. Please
00052  *    refer to the
00053  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6421.pdf">
00054  *    SAM-BA User Guide</a>, the
00055  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00056  *    GNU-Based Software Development</a> application note or to the
00057  *    <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00058  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00059  * -# On the computer, open and configure a terminal application
00060  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00061  *   - 115200 bauds
00062  *   - 8 bits of data
00063  *   - No parity
00064  *   - 1 stop bit
00065  *   - No flow control
00066  * -# Start the application.
00067  * -# In the terminal window, the following text should appear:
00068  *     \code
00069  *     -- USB Device HID Keyboard Project xxx --
00070  *     -- AT91xxxxxx-xx
00071  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00072  *     \endcode
00073  * -# When connecting USB cable to windows, the
00074  *    new "HID Keyboard Device" appears in the
00075  *    hardware %device list.
00076  * -# Once the device is connected and configured, pressing any of the board buttons
00077  *    should send characters to the host PC. Pressing num. lock should also make the third
00078  *    LED toggle its state (on/off).
00079  *
00080  * \section References
