00032  * The DDR/SDR SDRAM Controller (DDRSDRC) is a multiport memory controller. It comprises
00033  * four slave AHB interfaces. All simultaneous accesses (four independent AHB ports) are interleaved
00034  * to maximize memory bandwidth and minimize transaction latency due to SDRAM protocol.
00035  * 
00036  * \section ddr2 Configures DDR2
00037  *
00038  * The DDR2-SDRAM devices are initialized by the following sequence:
00039  * <ul>
00040  * <li> EBI Chip Select 1 is assigned to the DDR2SDR Controller, Enable DDR2 clock x2 in PMC.</li>
00041  * <li> Step 1: Program the memory device type</li>
00042  * <li> Step 2:
00043  *  -# Program the features of DDR2-SDRAM device into the Configuration Register.
00044  *  -# Program the features of DDR2-SDRAM device into the Timing Register HDDRSDRC2_T0PR.
00045  *  -# Program the features of DDR2-SDRAM device into the Timing Register HDDRSDRC2_T1PR.
00046  *  -# Program the features of DDR2-SDRAM device into the Timing Register HDDRSDRC2_T2PR. </li>
00047  * <li> Step 3: An NOP command is issued to the DDR2-SDRAM to enable clock. </li>
00048  * <li> Step 4:  An NOP command is issued to the DDR2-SDRAM </li>
00049  * <li> Step 5: An all banks precharge command is issued to the DDR2-SDRAM. </li>
00050  * <li> Step 6: An Extended Mode Register set (EMRS2) cycle is  issued to chose between commercialor high  temperature operations.</li>
00051  * <li> Step 7: An Extended Mode Register set (EMRS3) cycle is issued to set all registers to 0. </li>
00052  * <li> Step 8:  An Extended Mode Register set (EMRS1) cycle is issued to enable DLL.</li>
00053  * <li> Step 9:  Program DLL field into the Configuration Register.</li>
00054  * <li> Step 10: A Mode Register set (MRS) cycle is issued to reset DLL.</li>
00055  * <li> Step 11: An all banks precharge command is issued to the DDR2-SDRAM.</li>
00056  * <li> Step 12: Two auto-refresh (CBR) cycles are provided. Program the auto refresh command (CBR) into the Mode Register.</li>
00057  * <li> Step 13: Program DLL field into the Configuration Register to low(Disable DLL reset).</li>

00070 /** \addtogroup sdram_module
00071  *
00072  * \section sdram Configures SDRAM
00073  *
00074  * The SDR-SDRAM devices are initialized by the following sequence:
00075  * <ul>
00076  * <li> EBI Chip Select 1 is assigned to the DDR2SDR Controller, Enable DDR2 clock x2 in PMC.</li>
00077  * <li> Step 1. Program the memory device type into the Memory Device Register</li>
00078  * <li> Step 2. Program the features of the SDR-SDRAM device into the Timing Register and into the Configuration Register.</li>
00079  * <li> Step 3. For low-power SDRAM, temperature-compensated self refresh (TCSR), drive strength (DS) and partial array self refresh (PASR) must be set in the Low-power Register.</li>
00080  * <li> Step 4. A NOP command is issued to the SDR-SDRAM. Program NOP command into Mode Register, the application must 
00081  * set Mode to 1 in the Mode Register. Perform a write access to any SDR-SDRAM address to acknowledge this command. 
00082  * Now the clock which drives SDR-SDRAM device is enabled.</li>
00083  * <li> Step 5. An all banks precharge command is issued to the SDR-SDRAM. Program all banks precharge command into Mode Register, the application must set Mode to 2 in the
00084  * Mode Register . Perform a write access to any SDRSDRAM address to acknowledge this command.</li>
00085  * <li> Step 6. Eight auto-refresh (CBR) cycles are provided. Program the auto refresh command (CBR) into Mode Register, the application must set Mode to 4 in the Mode Register.
00086  * Once in the idle state, two AUTO REFRESH cycles must be performed.</li>
00087  * <li> Step 7. A Mode Register set (MRS) cycle is issued to program the parameters of the SDRSDRAM
00088  * devices, in particular CAS latency and burst length. </li>
00089  * <li> Step 8. For low-power SDR-SDRAM initialization, an Extended Mode Register set (EMRS) cycle is issued to program the SDR-SDRAM parameters (TCSR, PASR, DS). The write
00090  * address must be chosen so that BA[1] is set to 1 and BA[0] is set to 0 </li>
00091  * <li> Step 9. The application must go into Normal Mode, setting Mode to 0 in the Mode Register and perform a write access at any location in the SDRAM to acknowledge this command.</li>
00092  * <li> Step 10. Write the refresh rate into the count field in the DDRSDRC Refresh Timer register </li>
00093 * </ul>
00094 */
