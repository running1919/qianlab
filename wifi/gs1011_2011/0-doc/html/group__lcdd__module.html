Detailed Description

The driver implements LCD configuration and drawing functions.

Purpose

Implement driver functions for LCD control and image display.

  • Implement basic LCD controler configuration.
  • Implement display functions for LCD layers.
  • Implement simple drawing functions.
  • Implement string display functions.

Usage

Uses following functions for LCD basic configuration and displaying:

  1. Uses LCDD_Initialize() to initialize the controller and LCD.
  2. LCDD_On() and LCDD_Off() is used to turn LCD ON/OFF.
  3. LCDD_SetBacklight() is used to change LCD backlight level.
  4. To display a image (BMP format) on LCD, LCDD_ShowBMPRotated() LCDD_ShowBMPScaled() and LCDD_ShowBMP() can be used.
  5. To change configuration for an overlay layer, the following functions can use:
    1. LCDD_EnableLayer(), LCDD_IsLayerOn(): Turn ON/OFF layer, check status.
    2. LCDD_SetPosition(), LCDD_SetPrioty(), LCDD_EnableAlpha(), LCDD_SetAlpha(), LCDD_SetColorKeying(): Change display options.
  6. Shortcuts for layer display are as following:
    1. LCDD_ShowBase(), LCDD_StopBase()
    2. LCDD_ShowOvr1(), LCDD_StopOvr1()
    3. LCDD_ShowHeo(), LCDD_StopHeo()
    4. LCDD_ShowHcr(), LCDD_StopHcr()
  7. Drawing supporting fucntions, for drawing canvas:
    1. LCDD_CreateCanvas(): Create blank canvas on specified layer for drawing on
    2. LCDD_SelectCanvas(): Select a displayer as canvas to drawing on
    3. LCDD_GetCanvas(): Get current selected canvas layer

For LCD drawing functions, refer to LCD Driver Simple Drawing.

For LCD string display, refer to LCD Driver Font Display.
