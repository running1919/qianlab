Detailed Description

Purpose

Implement driver functions for CAN operations.

Usage

Uses following functions for CAN operations.

  1. Uses CAND_Handler() as peripheral interrupt handler.
  2. Uses CAND_Init() to initialize the driver and peripheral.
  3. Uses CAND_Activate() to enable the CAN interface. Then check CAND_IsReady() to find when CAN is synchronized for data transfer.
  4. To send a CAN message you can:
    1. Initialize mailbox with CAND_ResetMailbox()
    2. Start data transfer with CAND_Transfer()
  5. You can also start CAN message by:
    1. Initialize transfer with CAND_ConfigureTransfer(). Several transfers linked to different mailbox can be initialized.
    2. After configuration, transfers can be enabled by CAND_StartTransfers() at one time.
  6. To check if the transfer is done, CAND_IsTransferDone() is used.

The following structs should be instanced for driver operations.

  1. sCand : Driver instance struct.
  2. sCandMbCfg : Mailbox configuration parameters list.
  3. sCandTransfer : Message transfer operation parameters list.
