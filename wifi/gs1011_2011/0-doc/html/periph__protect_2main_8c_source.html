00030 /**
00031  *  \page periph_protect Peripheral Protect Example
00032  *
00033  *  \section Purpose
00034  *
00035  *  The Peripheral Protect example demonstrates how to prevent a program from
00036  *  corrupting a PIO controller behavior.
00037  *
00038  *  \section Requirements
00039  *
00040  *  This package can be used with AT91SAM9xx5 evaluation kits.
00041  *
00042  *  \section Description
00043  *
00044  *  The application shows the protective mechanism of the PIO controller.
00045  *  The example enables or disables write-protection of PIOB user interfaces.
00046  *  When the write-protection is enabled, any write attempt to the 
00047  *  write-protected registers will be detected and the write operation aborts. So 
00048  *  the value of the register won't be modified. Besides, the Write Protect Status 
00049  *  Register will indicate the offset address of the register.
00050  *
00051  *  \section Usage
00052  *
00053  *  -# Build the program and download it inside the evaluation board. Please
00054  *     refer to the
00055  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6224.pdf">
00056  *     SAM-BA User Guide</a>, the
00057  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00058  *     GNU-Based Software Development</a>
00059  *     application note or to the
00060  *     <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00061  *     IAR EWARM User Guide</a>,
00062  *     depending on your chosen solution.
00063  *  -# On the computer, open and configure a terminal application
00064  *     (e.g. HyperTerminal on Microsoft Windows) with these settings:
00065  *    - 115200 bauds
00066  *    - 8 bits of data
00067  *    - No parity
00068  *    - 1 stop bit
00069  *    - No flow control
00070  *  -# Start application.
00071  *  -# In the terminal window, the
00072  *     following text should appear (values depend on the board and chip used):
00073  *     \code
00074  *      -- Peripheral Protect Example xxx --
00075  *      -- xxxxxx-xx
00076  *      -- Compiled: xxx xx xxxx xx:xx:xx --
00077  *     \endcode
00078  *  -# Press one of the keys listed in the menu to perform the corresponding 
00079  *     action.
00080  *
00081  *  \section References
00082  *  - periph_protect/main.c
00083  *  - pio.h
00084  *  - pio.c
00085  */
