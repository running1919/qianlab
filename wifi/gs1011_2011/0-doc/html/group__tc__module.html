Purpose

Interface for the The Timer Counter (TC) peripheral.

The TC driver provides the Interface to configure the Timer Counter (TC).

Usage

  • Optionally, use TC_FindMckDivisor() to let the program find the best TCCLKS field value automatically.
  • Configure a Timer Counter in the desired mode using TC_Configure().

  • Start or stop the timer clock using TC_Start() and TC_Stop().

For more accurate information, please look at the TC section of the Datasheet.
