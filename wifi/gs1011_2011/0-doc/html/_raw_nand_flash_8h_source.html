00030 /**
00031  * \page raw_nand_page RawNandflash
00032  * 
00033  * \section Purpose 
00034  * 
00035  * RawNandflash is a bl Nandflash driver, it directly interacts with hardware's register to 
00036  * operate Nandflash interface, and it is called by upper layer drivers, such as EccNandFlash 
00037  * 
00038  * \section Usage 
00039  * 
00040  * -# RawNandFlash_Initialize() is used to initializes a RawNandFlash instance based on the given 
00041  *      model and physical interface. If no model is provided, then the function tries to autodetect 
00042  *      it. 
00043  * -# RawNandFlash_Reset() is used to reset a Nandflash device. 
00044  * -# RawNandFlash_ReadId() is used to read a Nandflash's id. 
00045  * -# RawNandFlash_EraseBlock() is used to erase a certain Nandflash device's block. 
00046  * -# RawNandFlash_ReadPage() and RawNandFlash_WritePage is used to do read/write operation. 
00047  * -# RawNandFlash_CopyPage() is used to issue copypage command to Nandflash device. 
00048  * -# RawNandFlash_CopyBlock() calls RawNandFlash_CopyPage to do a Nandflash block copy. 
00049 */
