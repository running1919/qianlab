Detailed Description

Purpose

Methods and definitions for configuring interrupts.

Usage

  1. Configure an interrupt source using IRQ_ConfigureIT
  2. Enable or disable interrupt generation of a particular source with IRQ_EnableIT and IRQ_DisableIT.
  3. Start or stop the timer clock using TC_Start() and TC_Stop().

Configures an interrupt in the AIC. The interrupt is identified by its source (ID_xxx) and is configured to use the specified mode and interrupt handler function. Mode is the value that will be put in AIC_SMRx and the function address will be set in AIC_SVRx. The interrupt is disabled before configuration, so it is useless to do it before calling this function. When AIC_ConfigureIT returns, the interrupt will always be disabled and cleared; it must be enabled by a call to AIC_EnableIT().
