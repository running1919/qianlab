Detailed Description

Here lists the control codes for SD/MMC low level driver to support lib.

  • SDMMC_IOCTL_POWER: Power ON/OFF control.
  • SDMMC_IOCTL_SET_CLOCK: Set clock and return actual clock.
  • SDMMC_IOCTL_GET_BUSMODE: Return max supported bus mode.
  • SDMMC_IOCTL_GET_HSMODE: Return if HS mode is supported.
  • SDMMC_IOCTL_GET_BOOTMODE: Return if Boot mode is supported.
