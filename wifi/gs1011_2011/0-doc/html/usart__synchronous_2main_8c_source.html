00029 /**
00030  *  \page usart_synchronous USART Synchronous Mode Example
00031  *
00032  *  \section Purpose
00033  *
00034  *  This example demonstrates the Synchronous mode provided by the USART
00035  *  peripherals on AT91SAM9xx5.
00036  *
00037  *  \section Requirements
00038  *
00039  *  This example can be used on AT91SAM9xx5 evaluation kit and USART0 and USART2 pins
00040  *  should be connected as following matching table:
00041  *   - <b>master   --  slave</b>
00042  *   -  TXD0(PA0)  --  RXD2(PA8)
00043  *   -  RXD0(PA1)  --  TXD2(PA7)
00044  *   -  SCK0(PA4)  --  SCK2(PB2)
00045  *
00046  *  \section Description
00047  *
00048  *  This application gives an example of how to use USART in synchronous mode.
00049  *  Synchronous operations provide a high speed transfer capability. The
00050  *  transfer under this mode needs a pair of master and slave, which is
00051  *  determined by which one offers the clock source.
00052  *
00053  *  The example initialized USART0 as master by default. To enable the
00054  *  communication between each side of the connection,the user should change
00055  *  the mode of another side to slave through user interface. If well configured,
00056  *  transfer could be started by typing 'r' and 'w' from terminal application.
00057  *  This example also leaves the interface to select the clock frequency.
00058  *
00059  *  The meaning of each input character could be found in items of the main menu.
00060  *
00061  *
00062  * \section Usage
00063  *
00064  *  -# Build the program and download it inside the evaluation board. Please
00065  *     refer to the <a href="http://www.atmel.com/dyn/resources/prod_documents/6421B.pdf">SAM-BA User Guide</a>,
00066  *     the <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">GNU-Based Software Development</a>
00067  *     application note or to the <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">IAR EWARM User Guide</a>,
00068  *     depending on your chosen solution.
00069  *  -# Connect a serial cable to the DBGU port on the evaluation kit.
00070  *  -# On the computer, open and configure a terminal application (e.g.
00071  *     HyperTerminal on Microsoft Windows) with these settings:
00072  *        - 115200 bauds
00073  *        - 8 data bits
00074  *        - No parity
00075  *        - 1 stop bit
00076  *        - No flow control
00077  *  -# Start the application. The following traces shall appear on the terminal:
00078  *     \code
00079  *     -- USART Synchronous Mode Example  xxx --
00080  *     -- AT91xxxxxx-xx
00081  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00082  *     -- Menu Choices for this example --
00083  *     -- [0-3]:Select clock frequency of master --
00084  *     -- i: Display configuration info
00085  *     -- w: Write data block .--
00086  *     -- r: Read data block.--
00087  *     -- s: Switch between master and slave mode.--
00088  *     -- m: Display this menu again.--
00089  *     --USART0 in MASTER mode--
00090  *
00091  *     \endcode
00092  *  -# The main menu will guide the user to configure the device and conduct
00093  *     operations.
00094  *
00095  *  \section References
00096  *  - usart_synchronous/main.c
