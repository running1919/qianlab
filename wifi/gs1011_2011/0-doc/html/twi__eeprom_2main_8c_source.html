00030 /**
00031  * \page twi_eeprom TWI EEPROM Example
00032  *
00033  * \section Purpose
00034  *
00035  * This basic example program demonstrates how to use the TWI peripheral
00036  * to access an external serial EEPROM chip.
00037  *
00038  * \section Requirements
00039  *
00040  * This package can be used with AT91SAM9xx5 evaluation kits.
00041  * Connect the TWI1 peripheral to EEPROM device:
00042  *        - <b>SAM9xx5-- EEPROM</b>
00043  *        - VCC -- VCC
00044  *        - GND -- GND
00045  *        - TWCK1(PA31) -- SCK
00046  *        - TWD1(PA30)  -- SCL
00047  *
00048  * All these pins can be easily connected with jumpers.
00049  *
00050  * When if works for twi_slave_example as a TWI master.
00051  * -# Connect TWD1 (SDA) for the 2 boards: PA31.
00052  * -# Connect TWCK1 (SCL) for the 2 boards: PA30.
00053  * -# Connect GND for the 2 boards: GND on extension header.
00054  * -# Add a pull up of 2,2KOhms on TWD and TWCK
00055  *
00056  * By default, PA30 and PA31 are SWJ-DP pins, it can be used as a pin for TWI1
00057  * peripheral in the end application. Mode selection between SWJ-DP (System IO mode)
00058  * and general IO mode is performed through the AHB Matrix CCFG_SYSIO.
00059  *
00060  * \section Description
00061  *
00062  * The code can be roughly broken down as follows:
00063  * <ul>
00064  * <li>Configure TWI pins.</li>
00065  * <li>Enable TWI peripheral clock.</li>
00066  * <li>Configure TWI clock.</li>
00067  * <li>Initialize TWI as twi master.</li>
00068  * <li>TWI interrupt handler.</li>
00069  * <li>The main function, which implements the program behavior.</li>
00070  * <ol>
00071  * <li>Set the first and second page of the EEPROM to all zeroes.</li>
00072  * <li>Write pattern in page 0. </li>
00073  * <li>Read back data in page 0 and compare with original pattern (polling). </li>
00074  * <li>Write pattern in page 1. </li>
00075  * <li>Read back data in page 1 and compare with original pattern (interrupts). </li>
00076  * </ol>
00077  * </ul>
00078  *
00079  * \section Usage
00080  *
00081  * -# Build the program and download it inside the evaluation board. Please
00082  *    refer to the
00083  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/6421B.pdf">
00084  *    SAM-BA User Guide</a>, the

00088  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00089  * -# On the computer, open and configure a terminal application
00090  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00091  *   - 115200 bauds
00092  *   - 8 bits of data
00093  *   - No parity
00094  *   - 1 stop bit
00095  *   - No flow control
00096  * -# Start the application.
00097  * -# In the terminal window, the following text should appear:
00098  *    \code
00099  *     -- TWI EEPROM Example xxx --
00100  *     -- AT91xxxxxx-xx
00101  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00102  *    \endcode
00103  * -# The following traces detail operations on the EEPROM, displaying success
