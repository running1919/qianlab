00030  * \page usart_serial USART Serial Example
00031  *
00032  * \section Purpose
00033  *
00034  * This example demonstrates the normal (serial) mode provided by the USART
00035  * peripherals.
00036  *
00037  * \section Requirements
00038  *
00039  *  This package can be used with AT91SAM9xx5 evaluation kits. 
00040  *
00041  * \section Description
00042  *
00043  * On start up, the debug information is dumpped to on-board DBGU port.
00044  * A terminal application, such as hyperterminal, is used to monitor these
00045  * debug information. Open another hyperterminal to connect with
00046  * on-board USART0 port. Then the program works in ECHO mode, so USART will
00047  * sends back anything it receives from the hyperterminal.  You can send a text
00048  * file from the hyperterminal connected with USART0 port to the device (without
00049  * any protocol such as X-modem).
00050  *
00051  *  \section Usage
00052  *
00053  *  -# Build the program and download it inside the evaluation board. Please
00054  *     refer to the <a href="http://www.atmel.com/dyn/resources/prod_documents/6421B.pdf">SAM-BA User Guide</a>,
00055  *     the <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">GNU-Based Software Development</a>
00056  *     application note or to the <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">IAR EWARM User Guide</a>,
00057  *     depending on your chosen solution.
00058  *  -# Connect a serial cable to the USART0 port on the evaluation kit.
00059  *  -# On the computer, open and configure a terminal application (e.g.
00060  *     HyperTerminal on Microsoft Windows) with these settings:
00061  *        - 115200 bauds
00062  *        - 8 data bits
00063  *        - No parity
00064  *        - 1 stop bit
00065  *        - No flow control
00066  *  -# Start the application. The following traces shall appear on the terminal:
00067  *     \code
00068  *     -- Basic USART Serial Project xxx --
00069  *     -- xxxxxx-xx
00070  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00071  *     -- Start to echo serial inputs --
00072  *     \endcode
