SD/MMC Driver Interface

Implementation for SD/MMC card SD/MMC mode physical layer driver. Supply a set of SD/MMC card SD/MMC mode's interface.

Usage

  • Instant SD/MMC struct and uses SD/MMC definitions in application (see SD/MMC Lib Commands) for SD/MMC management:
    1. sSdCard : Struct that defined by application for SD/MMC management.
    2. SD/MMC Card Types : SD/SDIO/MMC card type definitions.
    3. SD/MMC Return Codes : Return codes for SD/MMC operation (functions).
  • General Card Support: Initialize card with SD_Init() and then you can read/write on card.
    1. SD_Init(): Run the SDcard initialization sequence
    2. SD_GetCardType() : Return SD/MMC reported card type.
  • SD/MMC Memory Card Operations
    1. SD_ReadBlocks() : Read blocks of data
    2. SD_WriteBlocks() : Write blocks of data
    3. SD_Read() : Read blocks of data with multi-access command (Optimized read, see SD/MMC Read).
    4. SD_Write() : Read blocks of data with multi-access command (Optimized write, see SD/MMC Write).
    5. SD_GetNumberBlocks() : Return SD/MMC card reported number of blocks.
    6. SD_GetBlockSize() : Return SD/MMC card reported block size.
    7. SD_GetTotalSizeKB() : Return size of SD/MMC card in KBytes.
  • SDIO Card Operations: SD_Init() also detects SDIO card and then SDIO read/write functions can be used.
    1. SDIO_ReadDirect() : Read bytes from registers.
    2. SDIO_WriteDirect() : Write one byte to register.
    3. SDIO_ReadBytes() : Read 1 ~ 512 bytes from card.
    4. SDIO_WriteBytes() : Write 1 ~ 512 bytes to card.
