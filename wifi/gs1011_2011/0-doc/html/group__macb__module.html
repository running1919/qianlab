Detailed Description

Implement EMAC PHY driver, that initialize the PHY to prepare for ethernet transfer.

Usage

  1. EMAC related pins and Driver should be initialized at first.
  2. MAC address is set via EMAC_SetAddress().
  3. Initialize MACB Driver instance by invoking MACB_Init().
  4. Initialize PHY connected via MACB_InitPhy(), PHY address is automatically adjusted by attempt to read.
  5. Perform PHY auto negotiate through MACB_AutoNegotiate(), so connection established.
  6. Setup link speed by MACB_GetLinkSpeed() so link speed and duplex mode is desided.
  7. Now its time to send/receive ethernet packets via EMAC Driver
    • EMACD_Poll(): Polling received packets.
    • EMACD_Send(): Send a packet.
